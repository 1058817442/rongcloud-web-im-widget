# RongCloud Web IM Widget for Angular

---

**Web IM Widget for Angular** 是一个angular插件。通信部分依赖 `RongIMLib`。  
支持IE9+、Chrome、Firefox等

```
npm install -g typescript@1.6.0beta tsd grunt-cli bower
```

如有必要，使用 `sudo npm`

### 安装依赖库

在项目根目录下执行：

```
npm install
bower install
tsd install
```

### 编译开发代码

```
grunt build
```

### 发布正式代码

```
grunt release
```

### 编译demo

```
grunt demo
```

### 启动demo服务

```
grunt connect:demo
```
